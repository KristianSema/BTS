Esercizio BTS_1.1

1. A cosa serve bootstrap?

- Bootstrap include modelli di design basati su HTML e CSS per tipografia, moduli, pulsanti, tabelle, navigazione, modali, caroselli di immagini e molti altri, oltre a quelli opzionali. Bootstrap ti dà anche la possibilità di creare facilmente progetti reattivi.

2. Cosa devi importare per creare una pagina web con bootstrap?
<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js">
<script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.7/umd/popper.min.js">
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/js/bootstrap.min.js">

3. Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

- Se non vuoi scaricare e ospitare Bootstrap 4 da solo, puoi includerlo da un CDN (Content Delivery Network). MaxCDN fornisce supporto CDN per CSS e JavaScript di Bootstrap.

4. A cosa serve questo attributo del tag meta?

<content="width=device-width, initial-scale=1>

- Il width= device-width imposta la largezza della pagina per seguire la largezza dello schermo del dispositivo (che varierà a seconda del dispositivo). Il initial-scale= 1 imposta il livello di zona iniziale quando la pagina viene caricata per la prima volta dal browser.

5. Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)?

- Quando si utilizza .container-fluid poiché witdh è correlato alla larghezza della finestra, il display è dinamico Usa .container-fluid quando vuoi che la tua pagina si modifichi con ogni piccola differenza nelle dimensioni della finestra. Usa .container quando vuoi che la tua pagina cambi a solo 5 tipi di dimensioni, che sono anche conosciute come "punti di interruzione".
